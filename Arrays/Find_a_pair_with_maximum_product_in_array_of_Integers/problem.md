# Find a pair with maximum product in array of Integers

## Given an array with both +ive and -ive integers, return a pair with highest product

```txt
Examples :  

Input: arr[] = {1, 4, 3, 6, 7, 0}  
Output: {6,7}  

Input: arr[] = {-1, -3, -4, 2, 0, -5} 
Output: {-4,-5}
```
